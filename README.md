# Crypto Price Alert Bot

Script en Python para leer datos de ventas desde un archivo CSV, procesar los totales utilizando `pandas` y generar un reporte limpio en formato PDF con `ReportLab`. 

Ideal para automatizar reportes internos o enviar resúmenes periódicos sin depender de herramientas pesadas de BI.

## Tecnologías
* Python 3.10+
* Pandas (Manejo y cálculo de datos)
* ReportLab (Generación de PDF)

## Configuración y uso

   git clone [https://github.com/BGN26/generador-de-reportes.git](https://github.com/BGN26/crypto-alert-bot.git)
